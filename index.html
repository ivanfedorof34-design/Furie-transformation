Анализ временных рядов (FFT) ? Приложение загружает сигнал из CSV-файла, строит график исходных данных и показывает спектр после быстрого преобразования Фурье.

? CSV-файл должен содержать числовые значения сигнала. Обычно используется один столбец с отсчётами временного ряда.

? Отправляет выбранный CSV-файл на сервер для чтения и подготовки данных к построению графика.
? Запускает быстрое преобразование Фурье. Кнопка станет активной после успешной загрузки файла.

Исходный сигнал ? Здесь отображаются исходные отсчёты сигнала из CSV-файла. По оси X обычно идёт индекс отсчёта, по оси Y — значение сигнала.

Спектр частот ? Здесь показывается результат FFT. Выраженные пики обычно соответствуют частотам, которые сильнее всего присутствуют в исходном сигнале.
